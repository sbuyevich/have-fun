# Task 06: Verify Live Player Updates

## Goal

Verify that all open player screens update when a round starts.

## Work

- Open two player browser tabs or devices.
- Confirm both show waiting state.
- Start a round from the Master Dashboard.
- Confirm both player screens update without manual refresh.
- Confirm both players see the same shuffled word order.
- Start another round and confirm both players update again.

## Done Criteria

- All joined player screens update after Start Round.
- All players see the same shuffled words for the current round.
- Timer appears for each player after start.
- Player gameplay, submissions, correctness, and ranking remain out of scope.

## Verification

- Run the web app and test with at least two player tabs plus one master tab.
