# Task 04: Build Dashboard Result Grid

## Goal

Show ranked results on the Master Dashboard.

## Work

- Update `Dashboard.razor` and `Dashboard.razor.cs`.
- Show the correct sentence for the current round.
- Add a MudBlazor table or grid for results.
- Include rank, player name, submitted sentence, correctness, and spent time columns.
- Show an empty state before submissions.
- Keep dashboard master-only.

## Done Criteria

- Dashboard shows correct sentence.
- Dashboard shows each submitted player's sentence.
- Dashboard shows correctness.
- Dashboard shows spent time.
- Dashboard shows rank.
- Dashboard does not rank unsubmitted players.

- Build the solution from `src`.
