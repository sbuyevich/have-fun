# Task 01: Add Result Models

## Goal

Add reusable Core models for ranked round results.

## Work

- Add a `PlayerResult` model.
- Include rank, player name, submitted sentence, correctness count, total word count, spent time, and submitted timestamp.
- Add a `RoundResults` model if useful.
- Keep models in `HaveFun.Core`.
- Do not add persistence or cross-round history.

## Done Criteria

- Core has models that can represent current-round ranked results.
- Models contain all dashboard fields required by Stage 6.
- No database or persistent history is added.

## Verification

- Build the solution from `src`.
