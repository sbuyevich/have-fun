# Task 04: Add Sentence Selector

## Goal

Let the Game Master choose a predefined sentence from the dashboard.

## Work

- Read predefined sentences from `ISentenceLibrary`.
- Add a MudBlazor sentence selector.
- Show enough sentence text for the Game Master to choose.
- Show selected sentence preview.
- Show selected sentence time limit.
- Disable Start Round until a sentence is selected.
- Do not add custom sentence editing.

## Done Criteria

- Dashboard lists all predefined sentences.
- Selecting a sentence updates the preview.
- Selected sentence time limit is visible.
- Start Round is unavailable until a sentence is selected.

## Verification

- Build the solution from `src`.
- Join as master.
- Confirm all configured sentences appear.
- Select a sentence and confirm the preview updates.
